# LibasLounge# 🛍️ Libas Lounge

A modern, minimal, and Apple-style eCommerce website built using HTML, CSS, JavaScript, and Firebase.

---

## ✨ Features

- 🛒 Add to Cart system
- ➕ Quantity control & remove items
- 💳 Cash on Delivery (COD) checkout
- 📦 Order management system
- 🔍 Order tracking by phone number
- 🧑‍💼 Admin panel (add products & update order status)
- 📱 WhatsApp order confirmation
- 🎉 Clean success screen after order

---

## 🎨 Design

- Apple-style minimal UI
- Smooth and clean layout
- Mobile-friendly design
- Modern product cards

---

## 🔥 Tech Stack

- HTML
- CSS (Apple-style UI)
- JavaScript
- Firebase (Firestore Database)
